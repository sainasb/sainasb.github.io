To style the above, we need to add two tags together, We call this CSS Combinator.

What are Pseudo Classes?

Again, if we tihnk of CSS in terms of selectors, properties and values, then pseudo class is an add-on to a selector

            selector:psuedo_class {property: value;}

We use them for Links

If we want our link to look different, we can simply and the following

            a {color: red;}

But if we want each of our link states to look different then we will use pseudo class like the following:

  • :hover
  • :active
  • visited

But we can also make a new link look different.
