CSS

To get this, we need to to first define what we want our text to look like. In the following, this is what goes into style for what our text should look like.

h2 {
position: relative;
color: hsl(0, 0%, 68%);
font-family: "ofelia-text", sans-serif;
overflow: hidden;
border-right: 2px solid hsl(0, 0%, 80%);
white-space: nowrap;
animation: typewriter 4s steps(44) 1s 1 normal both, blinkTextCursor 500ms infinite;
}

KeyFrames


However, just this won't do anything. We also need to implement the keyframes in our style so that it actually has an animation:

@keyframes typewriter {
from {
width: 0;
}
to {
width: 1000px;
} }


@keyframes blinkTextCursor {
from {
border-right-color: hsl(0, 0%, 80%);
}
to {
border-right-color: transparent;
} }
